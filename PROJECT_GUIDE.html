קח כיוון · Kach Kivun

A full-stack web app that helps Israeli pre-army candidates find the right IDF role: profile onboarding, AI-driven role matching, an AI counselor & secretary, PDF reports, and an admin panel. This guide walks through every piece of the stack — frontend, backend, database, auth (including OTP), AI, and deployment.

1At a Glance

Three layers: a React app in the browser, an Express API in the cloud, and MongoDB for storage. Email OTP for sign-in, OpenAI for AI features, PDFKit for downloadable reports.

React 19 + TypeScript single-page app with file-based routing and Tailwind UI.

Node.js REST API built with Express, JWT auth, and Mongoose models.

DB Database

MongoDB Atlas (cloud). 7 collections covering users, OTPs, prefs, reports, AI usage, refresh tokens.

AI Intelligence

6Authentication — Including OTP

Email-first auth. The user types their email, gets a 6-digit code, and is signed in. The server issues a short-lived access token (JWT) and a refresh token that lives longer.

OTP sign-in flow

1
User enters emailFrontend posts to POST /api/auth/request-otp
2
Server generates a 6-digit codeSaved in EmailOtp with short expiry, sent via Nodemailer SMTP
3
User types the codeThe input-otp component captures 6 digits → POST /api/auth/verify-otp
4
Server verifiesMatches code + expiry, deletes the OTP row, finds or creates the User
5
Tokens issuedAccess JWT (15 min) + Refresh token (30 days, stored in DB)
6
Frontend stores tokensReact Query caches the session; all later calls send Authorization: Bearer …

Token lifecycle

A
Access token (JWT)Short-lived (~15 min). Signed with JWT_SECRET, contains user id + role.
B
Refresh tokenStored in RefreshToken collection. Used to mint a new access token without re-login.
C
ExpiryWhen access expires, frontend hits POST /api/auth/refresh automatically.
D
LogoutPOST /api/auth/logout deletes the refresh token from the DB.
E
Rate limitingAuth routes are protected by authLimiter — extra strict to block brute force.
F
Password pathOptional POST /api/auth/login with bcrypt-hashed password also works.

7The AI Layer

Three AI surfaces, all backed by the openai SDK. Every call is logged in AiUsageLog and capped by enforceTokenCap middleware so a single user can't burn the whole budget.

Role Matcher

Takes the user's Preferences + MilitaryStats and returns ranked IDF role matches with reasons.

AI Counselor

Conversational coach. Persists short context, answers questions about the army track.

AI Secretary

Helps with reminders, summaries, and document-style tasks.

10Mini Glossary (for explaining out loud)

JWT

A signed token the server gives the browser to prove who you are without re-checking the database every request.

OTP

"One-Time Password" — a short code sent to your email; valid once, for a few minutes.

Refresh token

A longer-lived token stored server-side that lets you get a new JWT silently.

Mongoose

Library that gives MongoDB documents a typed schema and helpers.

Vite

The dev server + bundler — instant reload while coding, optimized output for prod.

TanStack Router

Type-safe file-based router for React, replaces React Router here.

React Query

Caches and re-fetches server data so components stay in sync automatically.

Radix UI

Unstyled accessible components (modals, dropdowns) we style with Tailwind.

Tailwind

Utility CSS — write classes like px-4 text-lg instead of custom CSS files.

PDFKit

Node library that draws PDF files on the server, byte by byte.

Render

Cloud host that runs the Node API from the GitHub repo.

Cloudflare Workers

Edge platform that serves the React app fast, worldwide.

# IDF photo credits

All images in this folder are used under **Creative Commons Attribution-ShareAlike 3.0** from the IDF Spokesperson's Unit archive on [Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:Photographs_provided_by_the_IDF_Spokesperson%27s_Unit).

Official releases credit the photographer as **יחידת דובר צה״ל** (IDF Spokesperson's Unit photographer). Per-image scene descriptions and links are in `src/lib/idf-photo-catalog.ts`.

To add a new photo:

1. Download a CC-licensed file from Wikimedia (IDF Spokesperson category).
2. Save as `.jpg` in this folder.
3. Register `id`, `alt`, `credit`, `creditShort`, `commonsUrl`, and `keywords` in `idf-photo-catalog.ts`.
